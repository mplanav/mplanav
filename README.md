## 🚀 Featured Projects

### [LifeOS — Personal AI Agent](https://github.com/mplanav/LifeOS)
Personal AI agent built on MCP connecting Claude to 6 specialised servers and
31 tools: Gmail, Google Calendar, filesystem, smart notes (semantic search),
web search and daily briefings. Implements a server-to-server MCP pattern where
the briefing server acts simultaneously as MCP server and client.
`Python` `MCP` `Claude` `FastAPI` `Next.js` `OAuth2` `SQLite`

### [On-Demand Vehicle Routing](https://github.com/mplanav/pathfinding)
REST + WebSocket API implementing D* Lite pathfinding for dynamic vehicle
routing on grid maps with real-time obstacle updates. Supports up to 10
simultaneous start positions with automatic closest-point selection.
`FastAPI` `Docker` `WebSockets` `D* Lite` `asyncio`

### [Research Assistant CLI](https://github.com/mplanav/AI-agent-from-scratch)
LLM-powered CLI agent using LangChain that searches DuckDuckGo and Wikipedia,
synthesises results and returns structured Pydantic output with sources.
`Python` `LangChain` `DuckDuckGo` `Wikipedia` `Pydantic`

### [Discyn — Gym Mobile App](https://github.com/mplanav/discyn)
Full-stack cross-platform mobile app for gym tracking. FastAPI backend with
SQLAlchemy ORM, PostgreSQL database, Expo deployment.
`React Native` `FastAPI` `PostgreSQL` `SQLAlchemy` `Expo`

### [RBM Movie Recommender](https://github.com/mplanav/movie-recommender-rbm)
Recommendation engine using Restricted Boltzmann Machines trained with
Contrastive Divergence on MovieLens 100K.
`PyTorch` `Pandas` `NumPy`
